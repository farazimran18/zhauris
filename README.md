# Zhauris: FPGA-Based Cavitation Monitoring (Nexys 4 DDR)

Real-time **FPGA** monitor for **HIFU cavitation-like events** using multi-channel acoustic recordings.
The system detects broadband spectral energy bursts and estimates **Left / Center / Right** direction using **TDOA**, then visualizes results on **VGA** and prints runtime telemetry over **UART**.
